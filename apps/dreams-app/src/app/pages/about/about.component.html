Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda.

Dreams

Dreams (Name pending)™ is een forum website waar gebruikers berichten kan plaatsen over dromen of wensen die zij hebben. Hierbij kunnen zij hulp krijgen van andere gebruikers om deze droom waar te maken. Deze dromen kunnen groot of klein zijn bijvoorbeeld wil een gebruiker een bakkerij beginnen maar weet niet precies hoe en plaatst een bericht op Dreams (Name pending)™. Andere gebruikers reageren hierop met belangrijke bronnen en stappen over het beginnen van een bedrijf en het bakken van brood.

Entiteiten

Hier geef je een toelichting op je entiteiten. Voeg ook een afbeelding van je model toe zoals dat bij NoSql gevraagd wordt.

User

  • id: string
  • email: string
  • username: string
  • firstname: string
  • lastname: string
  • phonenumber: string
  • registerdate: date
  • password: string
  • network: users

Organisation

  • id: string
  • organisationcode: string
  • name: string
  • phonenumber: string
  • registerdate: date
  • password: string
  • verified: boolean
  • members: users

Post

  • id: string
  • titel: string
  • img: blob
  • vid: blob
  • context: string
  • readAvailability: int
  • tag: tags -> name: string, color: string
  • Poster: User OR Organisation

Reaction

  • id: string
  • vote: boolean
  • context: string
  • reactiondate: date
